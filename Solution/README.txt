# Instructions to navigate the solution folder

1. Folder code - contains the STATA DO files. Change the global directory to match the location on your local machine

2. Folder data - divided into three, namely raw_data, intermediate, and final
		2.1. raw_data folder contains all the raw data
		2.2. intermediate folder contains the modified dataset 
		2.3. final contains the final dataset used for the analysis

3. Folder literature - contains the literature used consulted to finalize task1 and task2

4. Folder output - contains the output generated while solving the exercises
